# Tape corpus

Verbatim `.tape` files collected 2026-07-09 from public GitHub repositories, kept
here as the measurement basis for autotape's profile defaults and lint thresholds
(aggregated in `../extracted-defaults.json`). Each file is named
`<owner>-<repo>_<path>.tape`. All credit to the original authors:

charmbracelet/vhs · dlvhdr/diffnav · byawitz/ggh · blacknon/hwatch ·
danvergara/dblab · Chleba/netscanner · bakks/butterfish · raitonoberu/sptlrx ·
mrusme/neonmodem · siddhantac/puffin · ajayd-san/gomanagedocker

These files are configuration scripts included for provenance; they are not part
of the published npm package.
